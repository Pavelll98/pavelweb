## Recommendations

- Export project thumbnails as **WebP** for best performance (add a `.jpg` fallback if needed)
- Keep `og-cover.jpg` under **200 KB**
- Keep portfolio thumbnails under **100 KB** each
- All images already have descriptive `alt` attributes in the HTML
